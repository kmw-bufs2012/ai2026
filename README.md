# 티타의 AI 2026 트렌드 연구소

김덕진 《AI 2026 트렌드&활용백과》(스마트북스, 2025.10)의 핵심을 **카드 + 퀴즈**로
압축 학습하는 웹 앱입니다. 게임 '하늘의 궤적'의 발명가 소녀 **티타**와 자동인형
**오르빌**이 가이드가 되어, 책 목차(1~8장 + 에필로그) 기반 전 챕터를 짧고 강한
카드로 안내합니다.

## 수록 범위 (책 목차 기준)

- **1장** 트렌드 키워드 10 (Trend 1~10)
- **2장** 범용 AI 마스터 (챗GPT-5 · 클로드 · 제미나이 + 공통 기능)
- **3장** 더 좋은 답변을 부르는 프롬프트 작성법
- **4장** 내 PC에 오픈소스 AI 설치·활용
- **5장** 검색 특화형 AI (퍼플렉시티 · 펠로 · oo.ai · 컨센서스)
- **6장** 에이전트 특화형 AI (젠스파크 · 스카이워크 · 마누스)
- **7장** 사무 특화형 AI (감마 · 미리캔버스 · 냅킨 · 노트북LM · 노션)
- **8장** 미디어 특화형 AI (미드저니 · 리크래프트 · 수노 · 힉스필드)
- **에필로그** AI 시대에 필요한 것, 회복 탄력성

## 컨셉 매핑

| UI 명칭            | 의미            |
| ------------------ | --------------- |
| 티타의 발명 노트   | 챕터 요약 카드  |
| 오르빌의 브리핑    | 핵심 3줄 요약   |
| 티타의 시험대      | 챕터별 퀴즈      |

## 기술 스택

- **Next.js 15** (App Router) + React 19 + TypeScript
- **Tailwind CSS** (기본 유틸 클래스만 사용)
- 별도 백엔드 없음 · 정적 데이터 기반
- 진도 · XP · 스트릭 · 뱃지는 **localStorage**에만 저장
- 배포 대상: **Vercel**

## 게이미피케이션

- **XP**: 카드 학습 +10, 퀴즈 정답 +20
- **레벨**: 누적 XP 100당 1레벨 (상단 진행 게이지)
- **스트릭**: 하루 1개 이상 학습 시 연속일 카운트
- **뱃지**: 첫 발명 노트 / 3일 연속 / 한 장 마스터 / 전 챕터 클리어

## UX 원칙 (태블릿 우선 · ADHD 친화)

- 갤럭시 탭 기준 최적화, 터치 영역 최소 44px
- 카드 하나 = 개념 하나, 짧은 문장·불릿
- 상단 진행 게이지 + "3/13" 카운트로 진행 상태 상시 표시
- 카드마다 다음 행동 버튼 1개를 크고 분명하게
- 미학습 / 학습중 / 완료를 색으로 구분

## 실행

```bash
npm install
npm run dev     # 개발 서버 (http://localhost:3000)
npm run build   # 프로덕션 빌드
npm run start   # 프로덕션 서버
```

## 톤앤매너 가이드 (매운맛 카피 원칙)

뻔한 "AI가 미래를 바꿉니다" 류는 금지합니다. 모든 카피는 **대담·직설 훅**으로
시작합니다. 티타는 냉정한 팩트와 뜨거운 트렌드 감각을 섞어 당돌하게 말하고,
오르빌은 핵심만 간결하게 3줄로 정리합니다. 단, 혐오·비방·성적 표현·과장된
허위/기만 문구는 절대 쓰지 않습니다 — 자극은 톤에서 나오고, 정확성은 팩트에서
나옵니다.
